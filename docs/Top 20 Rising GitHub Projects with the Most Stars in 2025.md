# Top 20 Rising GitHub Projects with the Most Stars in 2025

\# webdev \# programming \# ai \# javascript

If you’re like me, you’re always on the lookout for fresh tools that can make coding smoother, faster, or just more fun. The open-source world on GitHub never sleeps, and some projects really take off, racking up stars as more developers discover their value. I’ve put together a list of 20 rising GitHub projects with huge star counts that are worth checking out in 2025 from automation and APIs to dev tools and databases. Whether you want to improve your workflow or try something new, these projects have something for every developer.

###  Why These Tools Matter

As developers, we rely on tools that don’t just work but actually make our lives easier. The projects listed here have all caught the community’s attention for their ability to simplify complex workflows, automate repetitive tasks, or enhance productivity in meaningful ways. Whether it’s through better code quality checks, faster debugging, or innovative ways to handle APIs and databases, these tools are shaping how modern development happens.

###  1\. Puppeteer (⭐90.3k)

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fkn31r9fkeigs8bmymu9r.png)

[Puppeteer](https://github.com/puppeteer/puppeteer) is a Node library that provides a high-level API to control Chrome or Chromium browsers for automation and testing.

**Key Features:**

* Headless browser automation with full Chromium support
* Powerful scraping and web testing capabilities
* Screenshot and PDF generation from web pages
* Supports modern JavaScript features and async/await
* Easy integration with testing frameworks like Jest or Mocha

###  2\. Hoppscotch (⭐71k)

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fzuj2xjcms8qp5hwoqhgg.png)

[Hoppscotch](https://github.com/hoppscotch/hoppscotch) is an open-source API development ecosystem focused on simplicity and speed.

**Key Features:**

* Web and desktop clients for API testing
* Real-time request history and collections management
* Support for REST, GraphQL, and WebSocket protocols
* Environment variables for dynamic request parameters
* Lightweight and fast user interface

###  3\. Localstack (⭐58.5k)

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2F74mfmfnvczh61a14g8ci.png)

[Localstack](https://github.com/localstack/localstack) provides a fully functional local AWS cloud stack for rapid development and testing.

**Key Features:**

* Emulates a wide range of AWS services locally
* Supports Lambda, S3, DynamoDB, SQS, SNS, and more
* Easy integration with CI/CD pipelines
* Runs inside Docker containers for portability
* Speeds up cloud-native app development without actual AWS costs

###  4\. OpenHands (⭐52.6k)

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fa2jbe5lcyuctzpvsarak.png)

[OpenHands](https://github.com/All-Hands-AI/OpenHands) offers a collection of open-source projects for healthcare and humanitarian applications.

**Key Features:**

* Modular design for easy customization
* Focus on accessibility and low-resource environments
* Supports mobile and web platforms
* Emphasizes privacy and security by design
* Built for scalable deployments in developing regions

###  5\. Streamlit (⭐38.7k)

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fa2jbe5lcyuctzpvsarak.png)

[Streamlit](https://github.com/streamlit/streamlit) is a framework to quickly build and share data apps with simple Python scripts.

**Key Features:**

* Easy-to-use API for interactive visualizations
* Supports widgets like sliders, buttons, and text inputs
* Automatic UI updates on code changes
* Integrates well with popular Python data libraries
* Deploy apps instantly with minimal setup

###  6\. ShellCheck (⭐37.2k)

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2F9wskwmxsn0kgteiq3odb.png)

[ShellCheck](https://github.com/koalaman/shellcheck) is a static analysis tool that detects bugs and issues in shell scripts.

**Key Features:**

* Detects syntax and semantic errors in Bash scripts
* Provides actionable warnings and suggestions
* Supports multiple shell dialects including sh and zsh
* Integration with editors and CI pipelines
* Helps enforce best scripting practices

###  7\. Files (⭐36.7k)

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fctpyp9rjn0j7q83a84qd.png)

[Files](https://github.com/files-community/Files) is an open-source file manager designed for speed and simplicity on Windows.

**Key Features:**

* Fluent design language for a modern UI
* Tabbed browsing and quick previews
* Powerful file operations with multi-select support
* Supports extensions and plugins
* Lightweight and easy to customize

###  8\. Appsmith (⭐36.6k) — Versatile low-code platform

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fox7o2k9hefqnz53vgbrb.png)

[Appsmith](https://github.com/appsmithorg/appsmith) is a low-code platform to build internal tools quickly and collaboratively.

**Key Features:**

* Drag-and-drop UI builder for widgets
* Integrates with databases and REST APIs
* Role-based access control for teams
* Supports custom JavaScript logic
* Built-in deployment and version control

###  9\. HTTPie CLI (⭐35.2k) — Intuitive HTTP client

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fnblyokk9tw3hfw432qbq.png)

[HTTPie](https://github.com/httpie/cli) is a user-friendly command-line HTTP client designed for testing APIs.

**Key Features:**

* Simple, readable syntax for HTTP requests
* Supports JSON, forms, file uploads, and authentication
* Syntax highlighting and formatted output
* Plugins for extending functionality
* Cross-platform support

###  10\. Lapce (⭐35.1k) — Lightweight and fast code editor

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fnuw966k6ftsr4h6e2np3.png)

[Lapce](https://github.com/lapce/lapce) is a fast, open-source code editor written in Rust. It offers native performance, a minimal UI, and built-in features like Vim mode and remote development support.

**Key Features:**

* Built in Rust for high performance and low resource usage
* WASI-based plugin system for safe, extensible development
* Remote development with SSH and workspace sync
* Modal editing support with built-in Vim mode
* Lightweight and fast startup compared to Electron-based editors

###  11\. GPT-Pilot (⭐32.6k) — AI-driven task automation

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fyazvl1swksnpt97wutcb.png)

[GPT-Pilot](https://github.com/Pythagora-io/gpt-pilot) is an open-source autonomous AI agent framework powered by GPT.

**Key Features:**

* Automates complex multi-step workflows
* Supports task chaining and decision making
* Integrates with APIs and external tools
* Customizable via prompts and plugins
* Enables autonomous code generation and testing

###  12\. Refine (⭐30.5k) — Flexible React data management

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fyazvl1swksnpt97wutcb.png)

[Refine](https://github.com/refinedev/refine) is a React-based framework for building admin panels and internal tools with ease.

**Key Features:**

* Supports CRUD operations out-of-the-box
* Built-in data providers for REST and GraphQL
* Extensible with custom components
* Responsive design for desktop and mobile
* Rich authentication and authorization support

###  13\. SurrealDB (⭐29k) — Innovative multi-model database

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2F3fkwxjjqhqw721t17see.png)

[SurrealDB](https://github.com/surrealdb/surrealdb) is a scalable, distributed document-graph database with a flexible query language.

**Key Features:**

* Combines document and graph database models
* ACID compliance with distributed architecture
* Built-in authentication and permissions
* Real-time subscriptions and event triggers
* Supports SQL-like query language

###  14\. it-tools (⭐28.4k) — Comprehensive IT automation

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2F3fkwxjjqhqw721t17see.png)

[It-tools](https://github.com/CorentinTh/it-tools) is a set of powerful CLI utilities designed to boost developer productivity.

**Key Features:**

* Multiple utilities like HTTP clients, file searchers, and watchers
* Cross-platform support
* Minimal dependencies for easy installation
* Designed for automation and scripting
* Fast and efficient performance

###  15\. Lighthouse (⭐28.9k) — Powerful website performance auditor

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fui5cg0okw9b3i6n6v5ep.png)

[Lighthouse](https://github.com/GoogleChrome/lighthouse) is an open-source tool for auditing web page quality, focusing on performance, accessibility, and SEO.

**Key Features:**

* Provides detailed reports on web performance metrics
* Audits for accessibility and best practices
* SEO checks to improve search engine ranking
* Can run in Chrome DevTools or via CLI
* Supports automated testing in CI/CD pipelines

###  16\. Tabby (⭐30.8k) — Modern terminal emulator

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fjo68xa4qot1rhgeb9maq.png)

[Tabby](https://github.com/TabbyML/tabby) is a modern terminal emulator that emphasizes usability and extensibility.

**Key Features:**

* Supports multiple tabs and split views
* Fully customizable themes and keyboard shortcuts
* Built-in SSH client with session management
* Plugin system for extended functionality
* Cross-platform support

###  17\. DevDocs (⭐36k) — Reliable offline API docs browser

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fjo68xa4qot1rhgeb9maq.png)

[DevDocs](https://github.com/freeCodeCamp/devdocs) combines multiple API documentations in a fast, organized interface for developers.

**Key Features:**

* Offline access with progressive web app support
* Quick search across dozens of API docs
* Clean, minimalistic UI for focused reading
* Keyboard shortcuts for efficient navigation
* Lightweight and fast loading

###  18\. Daily.dev (⭐19.3k) — Handy developer news aggregator

Daily.dev is a browser extension aggregating developer news, articles, and tutorials.

**Key Features:**

* Personalized news feed for developers
* Supports filtering by technology and topics
* Easy access to top developer content in one place
* Lightweight and non-intrusive UI
* Integrates with popular browsers

###  19\. D2 (⭐20.3k) — Elegant diagram scripting language

D2 is a modern diagram scripting language designed for creating clear, maintainable diagrams.

**Key Features:**

* Simple syntax for flowcharts, graphs, and diagrams
* Auto-layout engine for neat designs
* Supports exporting to SVG and PNG
* Open-source with active community support
* Integrates well with documentation workflows

###  20\. PowerTools (⭐28.5k) — Handy Swiss Army Knife for Developers

[](https://media2.dev.to/dynamic/image/width=800%2Cheight=%2Cfit=scale-down%2Cgravity=auto%2Cformat=auto/https%3A%2F%2Fdev-to-uploads.s3.amazonaws.com%2Fuploads%2Farticles%2Fvkkna01o95o6wxlootrs.png)

[**DevToys**](https://github.com/DevToys-app/DevToys) is an all-in-one offline toolbox designed to assist developers with everyday tasks—without needing to search the web or install multiple extensions. It brings together dozens of utilities in one sleek desktop app, reducing context switching and saving time.

**Key Features:**

* Text tools for JSON formatting, base64 encoding/decoding, hashing, and more
* Regex tester with real-time feedback
* UUID generator and GUID formatter
* JWT decoder for token inspection
* Handy encoders, converters, and comparison tools all offline

DevToys is especially useful for developers who work across languages or frequently deal with structured data. It’s like having a full toolbox right on your machine.

###  Conclusion / Final Thoughts

These projects prove there’s always something new to explore in the dev ecosystem. Whether it’s a slick code editor, a powerful API client, or an AI tool helping automate your work, these rising stars on GitHub can save you time and headaches. Dive in, experiment, and see which ones fit your style you might find your next favorite tool here. And remember, the best part about open source is contributing back and being part of a community pushing tech forward.
